IUSEVIMBTW WEBMAIL DOCUMENTATION

---------------------------------------------------------------------------- MESSAGE LIST VIEW

This view supports normal (Esc), Ex (:), visual (Shift+v|v), and search mode (/).

in Ex-mode:
:e <folder name> Open folder.
:e <email> Compose new email.
:(q|q!|wq|x|qa!) Logout.

In nomal mode:
gi Navigate to Inbox.
gd Navigate to Drafts.
gs Navigate to Sent.
gj Navigate to Spam.
gt Navigate to Deleted Items.
j Select next item.
k Select previoius item.
gg select first item.
G select last item.
<C-d> Jump half page down.
<C-u> Jump half page up.
<C-w>(h|l) Switch focus between email list and folder list.
Shift + V Start visual mode.
dd Move selected message(s) to Deleted items folder.
D Purge selected message(s).
u Mark selected message(s) read.
U Mark selected message(s) unread.
r Reply on selected message.
R Reply-all on selected message.
f Forward selected message.
/ Start search.


---------------------------------------------------------------------------- COMPOSE VIEW

Enter insert-mode (i) to start typing a message. Exist to normal mode to navigate with Vim-motions. Press ":" to start Ex-mode. Use visual, visual line and visual block mode while composing the message.

In Ex-mode:
:w Save draft.
:(x|wq|wq!|x!) Send message and switch to inbox folder.
:q Switch to mail list.
:to <email address> Append email to "to" field.
:to! <email address> Set "to" field.
:cc <email address> Append email to "cc" field.
:cc! <email address> Set "cc" field.
:bcc <email address> Append email to "bcc" field.
:bcc! <email address> Set "bcc" field.
:subj <textg> Append to "subject".
:subj! <textg> Set "subject".
:set (nu|number) Show line numbers.
:set (rnu|relativenumber) Show relative line numbers.
:set (nonu|nonumber) Hide line numbers.
:set (nornu|norelativenumber) Hide relative line numbers.

Normal (Esc), Insert (i), Visual (v|shift+v|cntr+v) modes are the same-ish as in your Vim.
